# MGPEB - Aurora Siger

Este projeto foi desenvolvido como parte de uma atividade acadêmica com o objetivo de simular um sistema de gerenciamento de pouso de módulos em uma base em Marte.

## 📌 Descrição

O sistema simula a chegada de diferentes módulos (habitação, energia, laboratório, logística e médico) e organiza o processo de pouso com base em critérios como:

- nível de combustível
- prioridade do módulo
- condições externas (clima e sensores)

A decisão de pouso é feita utilizando lógica booleana (AND / OR), simulando um sistema real de tomada de decisão.

---

## ⚙️ Funcionalidades

- Organização dos módulos em fila (queue)
- Ordenação por prioridade
- Simulação de condições externas
- Verificação de autorização de pouso
- Separação entre módulos pousados e em alerta
- Busca do módulo com menor combustível

---

## 💻 Tecnologias utilizadas

- Python (estrutura básica, sem bibliotecas avançadas)

---

## ▶️ Como executar

1. Baixe o arquivo `mgpeb.py`
2. Execute em um ambiente Python (VSCode, terminal ou outro)
3. O sistema irá simular automaticamente o processo de pouso

---

## 📂 Estrutura do projeto

- `mgpeb.py` → código principal do sistema
- `MGPEB_Aurora_Siger.ipynb` → notebook utilizado no Google Colab
- `README.md` → documentação do projeto

## 🧠 Observação

Os valores dos módulos são gerados de forma aleatória para simular diferentes cenários de pouso, tornando cada execução única.
